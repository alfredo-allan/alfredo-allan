# 🧙🏻‍♂️ Alfredo Allan | Fullstack Developer & Cybersecurity Student

### Sobre Mim
Desenvolvedor Full Stack e estudante de Cibersegurança focado em construir soluções robustas e seguras. Recentemente, conquistei a certificação **AWS Academy Graduate - Cloud Security Foundations** e concluí o curso de **Cibersegurança Essencial pela Cisco em parceria com o SENAI**, consolidando meus conhecimentos em defesa de redes e infraestrutura em nuvem.

No desenvolvimento, utilizo **Python** com os frameworks **Flask** e **FastAPI** para criar APIs eficientes, e **React** para construir interfaces modernas e dinâmicas. Meu objetivo é sempre unir a agilidade do desenvolvimento com as melhores práticas de segurança da informação.
